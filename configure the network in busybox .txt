✅ Modify your rcS (network only)

Edit:

nano ~/rootfs/etc/init.d/rcS
🔧 Add this at the END (before any shell/exit)
echo "[*] Starting network..."

# Bring interface up
ip link set eth0 up

# Get IP via DHCP
udhcpc -i eth0 -q
⚠️ CRITICAL (this is why it failed earlier)

You MUST have the DHCP script, otherwise:
👉 “lease obtained” but no IP applied

📁 Create this file:
mkdir -p ~/rootfs/usr/share/udhcpc
nano ~/rootfs/usr/share/udhcpc/default.script
Paste:
#!/bin/sh

case "$1" in
    deconfig)
        ip addr flush dev $interface
        ;;
    renew|bound)
        ip addr add $ip/$subnet dev $interface
        ip route add default via $router
        echo "nameserver 8.8.8.8" > /etc/resolv.conf
        ;;
esac

Make executable:

chmod +x ~/rootfs/usr/share/udhcpc/default.script
🔁 Repack initrd
cd ~/rootfs
find . -print0 | cpio --null -ov --format=newc | gzip -9 > ~/boot/initrd.img
▶️ Boot and test

After boot:

ip a

You should see:

inet 192.168.29.x
🧪 Test connectivity
ping 192.168.29.1
ping 8.8.8.8
